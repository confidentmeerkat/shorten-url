API Guide

With this API you can create a new short link, get a short link, get an original URL, or get all URLs and their short links.

Create a short link:

For creating a new short link, you should send a POST request to /api/create containing a JSON with an origin Key and the URL you want to be shortened.
Notice: you must send a URL with the schemes http:// or https:// anything else doesn't work.

Example:

{"origin": "https://google.com"}

Get an existing short link:

You should send a Get request to /api/get containing a query with origin as the key and URL as the value.

Example: 

example.com/api/get?origin=https://google.com

Get an original link:

You can send a shortened link and get its original link, for this purpose you should send a Get request to /api/get containing a query with short as the key and shortened link as the value.

Example: 

example.com/api/get?short=Iv1a

Get all URLs and their short links:

To get all URLs and their short links, you need to just visit the /api/all on the GET method.

Example: 

example.com/api/all
